## 资产字段规定
```
{
    type_name:str          	# AMOUNT or ITEM
    quantity:int            # 数量
    value:int               # 价值
    name:str				# 资产名称
    category:str            # 资产分类（注意不是id）
 	nid:                    # 资产的唯一标识
    histroy:list            # 历史
	description:str			# 资产描述
    parent:				 	# 唯一 用id标识
    children:list			# 用id标识
    owner:					# 用用户名表示
    department:				# 部门
    status:         		# 状态
    start_time:				# 录入时间
    quit_time:				# 清退时间
    prop:str				# 自定义属性
}
```
